WHY IT SOUNDS BETTER

No resample. No mixer. Just the bits.

Most players — Apple Music included — quietly resample everything to one system rate before it reaches your DAC. EKO sends the file's exact samples at the file's exact rate, and switches the device to match. Less in the path means more of the recording.

A 32-bit float volume stage and an opt-in EQ are there when you want them — and the seal tells you the moment the signal is no longer pristine.

GET STARTED

Clone, build, listen.

EKO is open source under the MIT license. Built with Tauri, Rust and React — a ~15 MB native app.

# grab it
$ git clone https://github.com/reactivepixels/eko.git && cd eko
$ npm install
# run it
$ npm run tauri dev
# or ship a .app
$ npm run tauri build
